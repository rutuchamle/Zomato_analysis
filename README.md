# Zomato Analysis Project

## Project Overview
This project is based on Zomato restaurant/order data analysis. The main aim of this project is to clean, analyze, and visualize Zomato data using **Python, SQL, and Power BI**.

The repository contains Python analysis notebook, cleaned dataset, SQL file, and Power BI dashboard file. 

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / Seaborn
- MySQL / SQL
- Power BI

## Files in This Repository
- `zomato.csv` - Original dataset
- `cleaned_zomato.csv` - Cleaned dataset
- `Python.ipynb` - Python data cleaning and analysis
- `zomato_sql.sql` - SQL queries for analysis
- `zomato_powerbi.pbix` - Power BI dashboard
- `README.md` - Project documentation

## Objectives
- Clean raw Zomato data
- Remove null values and duplicate records
- Analyze restaurant/order trends
- Find total revenue, average rating, and order patterns
- Identify top restaurants, popular food categories, and customer behavior
- Create interactive dashboard in Power BI

## Python Analysis
In Python, the dataset was cleaned and analyzed using Pandas. Important steps include:

- Loading dataset
- Checking missing values
- Removing duplicates
- Changing data types
- Creating useful columns
- Performing exploratory data analysis

## SQL Analysis
SQL was used to perform business-related queries such as:

- Total revenue
- Top restaurants by orders
- Average rating
- Most ordered food items
- Monthly and daily order trends
- Customer order behavior

## Power BI Dashboard
The Power BI dashboard shows:

- Total revenue
- Total orders
- Average rating
- Top restaurants
- Orders by day
- Orders by category
- Monthly sales trend
- Customer and location analysis

## Key Insights
- Identified high-performing restaurants
- Found popular food categories
- Analyzed order trends by day and month
- Studied customer ordering behavior
- Created visual insights for better decision-making
